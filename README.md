# Minecraft Tool Stats API

## Description
This project demonstrates how to retrieve and display Minecraft tool statistics using an API.  
The program allows users to input the name of a Minecraft tool and receive information such as durability, mining speed, and material type.  
It shows how Python can interact with APIs and display useful game data in a simple command-line interface.
